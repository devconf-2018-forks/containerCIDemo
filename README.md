# Container CI Demo
Container CI using Openshift and Jenkins pipelines presented at DevConf.cz 2018

# Demo requirements
- Instance of Openshift
- Deployment of Jenkins
- Jenkins instance with plugins:
 - Pipeline plugins
 - Kubernetes
 - Openshift Client
 - GitHub Pull Request Builder
